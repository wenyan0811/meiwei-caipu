# 🍳 美味菜谱

一个跨平台的菜谱应用，支持搜索、收藏和添加菜谱功能。

## 🌐 在线访问
📱 **GitHub Pages在线版**：[点击访问菜谱APP](https://您的用户名.github.io/meiwei-caipu/)

## ✨ 功能特点
- 🔍 智能搜索菜谱
- 📂 分类筛选（家常菜、川菜、粤菜等）
- ❤️ 收藏功能  
- ➕ 添加自定义菜谱
- 📱 完美适配移动端
- 🌐 跨平台支持
